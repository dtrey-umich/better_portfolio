# Design Inspiration & Direction

## Visual Style Goals
- Clean, Apple-inspired minimalism with prominent whitespace
- Bauhaus influence with bold geometric shapes
- Technical precision balanced with artistic freedom
- Hand-drawn elements to add warmth and personality
- Bold primary colors (red, yellow, blue) used strategically
- Fine black lines for structure and definition
- Soft, blended color gradients
- Rounded edges and organic shapes

## Color Palette
### Primary Colors
- Red: #FF3B30 (or your specific brand red)
- Yellow: #FFCC00 (or your specific brand yellow)
- Blue: #007AFF (or your specific brand blue)

### Color Blends
- Red to Yellow gradient
- Yellow to Blue gradient
- Blue to Red gradient
- Soft, airy color transitions
- Subtle color overlays

### UI Colors
- Background: Pure white (#FFFFFF)
- Secondary Background: Light gray (#F5F5F7)
- Text: Deep black (#1D1D1F)
- Accent Lines: Fine black (#000000)
- Interactive elements: Brand colors with gradients
- Success/Error states: System colors

## Typography
### Font Families
- Headings: A bold, geometric sans-serif (like Inter or SF Pro Display)
- Body text: Clean, readable sans-serif (like SF Pro Text)
- Accent text: A font with hand-drawn qualities (like Sketch or Marker Felt)
- Technical elements: Monospace font (like SF Mono)

### Type Scale
- H1: 48px / 56px line height
- H2: 36px / 44px line height
- H3: 24px / 32px line height
- Body: 16px / 24px line height
- Small text: 14px / 20px line height

## Design Principles
1. Technical Meets Artistic
   - Precise layout with expressive elements
   - Clean typography with hand-drawn accents
   - Systematic design with organic touches

2. User Experience
   - Intuitive interactions
   - Fast loading
   - Smooth transitions
   - Mobile-first approach

3. Accessibility
   - High contrast ratios
   - Clear typography
   - Keyboard navigation
   - Screen reader support

4. Performance
   - Optimized images
   - Efficient animations
   - Progressive loading
   - Responsive design
